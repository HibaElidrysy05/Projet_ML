Atelier : Régression Logistique

Machine Learning | Semestre 5

Informations sur le Dataset

  • Nom du dataset : Pima Indians Diabetes
  • Nombre d'exemples : 768
  • Nombre de variables : 8 attributs médicaux
  • Distribution classes : 500 non-diabétiques (0), 268 diabétiques (1)
  • Variable cible : Présence de diabète (0 ou 1)
  • Source : UCI Machine Learning Repository

Description

  • Utilisation d’un modèle de régression logistique
  • Normalisation des données
  • Gestion des valeurs aberrantes et zéros anormaux
  • Classification binaire : diabétique vs non-diabétique
  • Évaluation avec Accuracy, Precision, Recall, F1-score et Matrice de confusion
