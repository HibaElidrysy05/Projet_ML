Atelier : Régression Linéaire

Machine Learning | Semestre 5

Informations sur le Dataset

  • Nom du dataset : Taxi Trip
  • Nombre d'exemples : 1000
  • Valeurs manquantes : 50 valeurs manquantes réparties sur plusieurs colonnes
  • Nombre de variables : 10 variables explicatives
  • Variable cible : Trip_Price (prix final du trajet)
  • Source : Dataset interne (trajets de taxi)

Description

  • Utilisation d’un modèle de régression linéaire
  • Normalisation des données
  • Gestion des valeurs manquantes
  • Variables explicatives : distance parcourue, moment de la journée, jour de la semaine, nombre de passagers, trafic, météo, tarifs appliqués (base, prix/km, prix/min)
  • Évaluation avec MSE et R²
